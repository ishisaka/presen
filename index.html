基本操作：ハンズオン

あなたは新しいWebプロジェクトを開始(擬似的な)するにあたって Gitでソース管理をおこなうことにしました。 今からWebページを作ることにしましょう！ まず、コンソールを立ち上げます。

バージョンを確認

バージョンの表示を行います。
# git --version git version 1.7.11.msysgit.1
私のおまじない:export PS1="$(__git_ps1)# "

設定の設定と確認

# git config --global user.name xxxxxx # git config --global user.email xxxx@xxxx.com
# git config --list --global xxxxxx=xxxxx xxxxxx=xxxxx

リポジトリを作る

ディレクトリを作る ディレクトリに移動する
# mkdir handson # cd handson # git init Initialized empty Git repository in xxxx # ls .git
.gitが該当ディレクトリにできます。

ファイルをリポジトリに追加(登録)する

該当ディレクトリにindex.htmlを作成します。
# git add index.html

ステータスの確認

現在の変更状況を見ることができます。
# git status # On branch master # # Initial commit # # Changes to be committed: xxxxx
masterの説明

コミットを行う

# git commit -m "register index page"
